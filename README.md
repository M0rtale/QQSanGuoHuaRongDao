# QQ三国华容道暴力解法

运用了所谓的Breadth-First-Search，虽然特别慢而且很暴力，但能相对来说很快的解出顺序

支持任何x乘x的，顺序为{0,1,2,3,...,x}的图。输入任何非正方形图或者与大小不同会有未知错误。

使用方法： solve <size> [x1,x2,x3...xSize] [targ1,targ2,targ3...targsize]
